### 📖 Project Overview
The objective of this task was to design a **Low-Fidelity Wireframe** for an e-commerce platform named **Easy Cart** (Website/App). The focus of a low-fidelity wireframe is to establish the core structure, user flow, information architecture, and layout of the application without getting distracted by visual design elements like colors, typography, or exact imagery. 

**Easy Cart** aims to provide a seamless, intuitive, and frictionless shopping experience for users, focusing on clean navigation and a straightforward checkout pipeline.

## 🎯 Objectives Achieved

* **Foundational Wireframe Design:** Designed and mapped out foundational layouts for the core 7-screen e-commerce journey:
  * **Welcome Page:** Structuring a clean initial entry point to introduce the "Easy Cart" brand.
  * **Login Page:** Designing a frictionless user authentication screen with clear input states for credentials.
  * **Home Page / Dashboard:** Layout mapping for product categories, search bars, promotional banners, and personalized feeds.
  * **Product Page:** Establishing content hierarchy for item imagery, detailed specifications, pricing structure, and the primary "Add to Cart" CTA.
  * **Cart Page:** Creating an organized layout for reviewing selected items, editing quantities, and displaying order total breakdowns.
  * **Payment Page:** Structuring secure and intuitive form layouts for payment method selection (Card, UPI, Wallets) and billing details.
  * **Order Confirmation Page:** Designing a post-purchase success screen featuring order summary highlights and tracking details.
* **Information Architecture:** Established a seamless, linear navigation structure and user flow, minimizing cognitive load for shoppers from discovery to checkout.
* **Low-Fidelity Aesthetic:** Intentionally utilized a grayscale, minimal aesthetic to strictly emphasize functionality, structural spacing, and element layout over final visual polish.
