### 📌 Project Overview
This task serves as the final mini-project for the CodeAlpha UI/UX Design internship. The objective was to synthesize the work from Tasks 1 and 2 by creating a fully clickable, interactive prototype of the **Easy Cart** application.

### 🎯 Objectives Achieved
* Linked all high-fidelity screens in Figma to create a seamless, end-to-end user journey.
* Implemented functional navigation logic, including back buttons, tab switching, and active states.
* Demonstrated a complete visual and interactive representation of how the application would function in the hands of a real user.
