### 📖 About This Repository
Welcome to my official internship portfolio! This repository contains the design files, prototypes, and case studies I developed during my **UI/UX Design Internship at CodeAlpha**. 

The primary goal of this internship was to master the end-to-end design thinking process—from low-fidelity wireframing and user journey mapping to creating polished, high-fidelity interactive prototypes using modern visual principles.

*📸 **Media Included:** In addition to the direct Figma links, this repository also contains **video walkthroughs** and **high-resolution screenshots** for each task. These files are provided to give a quick, accessible preview of the final interfaces, component states, and prototype interactions directly from GitHub.*

#### 📊 [Task 3: UX Case Study (Swiggy Redesign)](#)
Conducted a comprehensive UX audit of the Swiggy mobile Food Delivery app. Analyzed the user journey, identified frictional weaknesses (like dark patterns and visual clutter), and proposed 3 actionable high-fidelity design improvements.
* **Report:** *Please see the attached `.docx` / `.pdf` file in this repository for the full 5-page case study and mockups.*
